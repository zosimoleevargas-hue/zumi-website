# Proyecto

## Nombre

Zumi Website

---

## Objetivo

Crear un sitio web moderno para Zumi que fortalezca su presencia digital, genere confianza, aumente las visitas al restaurante y facilite que los clientes realicen pedidos.

---

## Estado

🟡 Planeación

---

## Cliente

Zumi

---

## Tipo de negocio

Restaurante de comida saludable especializado en ensaladas, wraps y sándwiches.

---

## Objetivos del negocio

- Aumentar las ventas.
- Generar más pedidos.
- Mejorar la presencia digital.
- Aparecer mejor en Google.
- Mostrar el menú de forma clara.
- Facilitar el contacto por WhatsApp y teléfono.

---

## Stack Tecnológico (Planeado)

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React
- Vercel

---

## Filosofía del proyecto

Antes de escribir código se documenta el negocio, se define la arquitectura y se crea una especificación. La IA implementa el proyecto siguiendo esa documentación.
