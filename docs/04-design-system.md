# Design System

## Objetivo

Crear una identidad visual consistente, moderna y profesional para Zumi.

La experiencia debe transmitir frescura, salud, rapidez y confianza.

---

# Personalidad Visual

La interfaz debe sentirse:

- Limpia
- Moderna
- Fresca
- Cercana
- Natural
- Minimalista

No debe sentirse:

- Corporativa
- Elegante de lujo
- Oscura
- Saturada
- Infantil

---

# Inspiración

Referencia principal:

- Sweetgreen

Referencias secundarias:

- Apple
- Linear

No copiar el diseño.

Copiar la lógica.

---

# Paleta de Colores

## Primary

Verde

Representa:

- Salud
- Frescura
- Naturaleza

## Accent

Naranja

Representa:

- Energía
- Acción
- Hambre
- Llamadas a la acción

## Background

Blanco cálido

Debe transmitir limpieza.

## Surface

Gris muy claro.

Se utiliza para separar secciones.

## Text

Gris oscuro.

Nunca negro puro.

---

# Tipografía

Utilizar una fuente moderna.

Recomendación:

Inter

Reglas

- Títulos grandes.
- Excelente legibilidad.
- Mucho espacio entre líneas.
- No utilizar más de dos pesos por sección.

---

# Botones

## Primario

Color:

Naranja.

Texto blanco.

Acción principal.

Ejemplo:

"Pedir por WhatsApp"

---

## Secundario

Fondo blanco.

Borde verde.

Texto verde.

Ejemplo:

"Ver Menú"

---

# Cards

Todas las tarjetas deben mantener:

- Esquinas redondeadas.
- Imagen superior.
- Espaciado amplio.
- Sombra ligera.
- Hover suave.

---

# Iconografía

Utilizar una sola librería.

Recomendación:

Lucide React.

No mezclar estilos.

---

# Espaciado

La interfaz debe respirar.

Mucho espacio entre secciones.

Mucho espacio alrededor de los títulos.

No saturar información.

---

# Animaciones

Utilizar únicamente animaciones suaves.

Permitidas:

- Fade In.
- Slide Up.
- Hover.
- Scale.

Evitar animaciones innecesarias.

---

# Responsive

Diseñar Mobile First.

El botón de WhatsApp debe ser visible rápidamente desde dispositivos móviles.

---

# Fotografías

Las fotografías deben ser protagonistas.

Siempre utilizar imágenes reales del restaurante cuando estén disponibles.

Evitar fotografías genéricas de stock en la versión final.

---

# Principios de Diseño

Antes de aceptar cualquier decisión de diseño preguntarse:

¿Hace que el usuario quiera pedir comida?

¿Hace que el sitio sea más claro?

¿Reduce la cantidad de clics?

¿Genera confianza?
