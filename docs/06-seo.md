# SEO Strategy

## Objetivo

Optimizar el sitio web de Zumi para mejorar su visibilidad en Google, especialmente para búsquedas locales relacionadas con comida saludable en Mexicali.

---

# SEO Local

La prioridad principal es aparecer cuando usuarios busquen:

- Restaurante saludable en Mexicali.
- Comida saludable cerca de mí.
- Ensaladas en Mexicali.
- Wraps en Mexicali.
- Sándwiches saludables en Mexicali.

---

# Información del Negocio

Implementar datos estructurados de tipo:

Restaurant / LocalBusiness

Debe incluir:

- Nombre del negocio.
- Dirección.
- Teléfono.
- Horarios.
- Ubicación.
- Categoría del negocio.

---

# Metadata

Cada página debe tener:

## Title

Debe incluir:

- Nombre del negocio.
- Servicio principal.
- Ubicación.

Ejemplo:

Zumi | Comida saludable en Mexicali

---

## Description

Debe comunicar:

- Qué vende.
- Beneficio principal.
- Ubicación.

---

# Open Graph

Configurar para compartir correctamente en redes sociales.

Debe incluir:

- Título.
- Descripción.
- Imagen principal.
- URL.

---

# Imágenes

Todas las imágenes deben:

- Tener nombres descriptivos.
- Tener texto alternativo.
- Estar optimizadas.
- Usar formatos modernos cuando sea posible.

Ejemplo:

Correcto:

ensalada-saludable-zumi-mexicali.webp

Incorrecto:

IMG_38291.jpg

---

# Performance SEO

Priorizar:

- Carga rápida.
- Optimización de imágenes.
- Buen Core Web Vitals.
- Código limpio.

---

# Indexación

Configurar:

- sitemap.xml
- robots.txt
- Canonical URL

---

# Contenido

El sitio debe incluir contenido que Google pueda entender:

- Descripción del restaurante.
- Menú.
- Ubicación.
- Horarios.
- Servicios.
- Opiniones.

---

# Google Business Profile

El sitio debe complementar el perfil de Google Business.

Mantener consistencia en:

- Nombre.
- Dirección.
- Teléfono.
- Horarios.

---

# Objetivo final

Que una persona buscando comida saludable en Mexicali pueda encontrar Zumi, confiar en el negocio y realizar una acción:

- Visitar el restaurante.
- Llamar.
- Pedir por WhatsApp.
